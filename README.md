# 3D Portfolio

This is my first web development project—a beginner-friendly interactive portfolio built with HTML, CSS, and JavaScript.

## Features
- Interactive 3D room with animated doors
- Puzzle-based navigation to unlock content
- Custom cursor, ripple effects, and confetti celebration
- Beginner-friendly portfolio presentation

## How to use
1. Open `index.html` in your browser.
2. Click `START GAME`.
3. Solve the puzzles to unlock the `About`, `Projects`, and `Contact` sections.
